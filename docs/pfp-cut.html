Descrição

O Pfp-cut é uma ferramenta web leve, desenvolvida sem frameworks, que permite carregar imagens locais, posicionar e redimensionar uma área de recorte interativa e exportar o resultado em PNG ou BMP.

O projeto foi criado para explorar recursos nativos do navegador, incluindo manipulação de imagens, Canvas API, FileReader API e geração manual de arquivos BMP.

Configurações

Assim que o site é aberto, damos de cara com o recorte, ainda vazio. No canto superior direito temos algumas Configurações:

  • Tema | Altera entre escuro e claro
  • Arredondar | Alterna entre o recorte quadrado e o redondo
  • Renomear | Altera o nome do arquivo final [Se estiver vazio usa o nome original]
  • Resolução de saida | Define o valor de altura e largura da imagem final

Recorte

Ja partindo para a ação principal do site temos 3 opções.
Entrada -> Ajustar imagem
Saida -> Baixar .PNG e .BMP

Tela de recortes

Recorte

Nessa tela é possível executar algumas ações:

  • Canva > Exibe a imagem base no fundo, e o rastro do recorte, (parte mais clara), quando com recorte redondo, exibe a visão como no segundo slide
  • Arquivo > Procura a imagem base [.png, .jpeg, .jpg, .bmp]
  • Nome do arquivo > Exibe o nome original da imagem
  • Enviar > Envia a imagem exibida no canva para a tela inicial
  • Slider > Permite um controle mais preciso do zoom de recorte exibido no canva.
  • Mouse > Com o mouse podemos arrastar o recorte usando o ponteiro, ou aumentar o zoom usando o scroll

Resultado

Imagem recortada

Ao final do recorte, clicando em enviar, voltamos para a tela inicial.

A prévia que aparece no centro é a mesma que será salva, ou em BMP ou em PNG.

O ultimo passo é baixar a imagem, podendo sempre reajustar o zoom antes.

⚙️ Executando o projeto

Clone o repositório:

git clone https://github.com/Ntzzn-Dev/Pfp-cut.git

Após o download, basta abrir o arquivo index.html em qualquer navegador moderno.

🚀 Funcionalidades

  • Upload de imagens (.png, .jpg, .jpeg e .bmp)
  • Área de recorte interativa
  • Movimentação do recorte com o mouse
  • Zoom utilizando scroll ou slider
  • Recorte quadrado ou circular
  • Pré-visualização em tempo real
  • Exportação em PNG e BMP

📖 Como utilizar

  1. Escolha o formato do recorte (quadrado ou circular).
  2. Clique em Ajustar imagem.
  3. Selecione uma imagem do computador.
  4. Posicione o recorte arrastando com o mouse.
  5. Ajuste o zoom utilizando o scroll ou o slider.
  6. Clique em Enviar.
  7. Baixe o resultado em PNG ou BMP.
