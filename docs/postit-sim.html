Postit Simulator

Autor: Jhonatan | Nathan

GitHub: https://github.com/Ntzzn-Dev/postit-sim

Data: 25/04/2026

Descrição

Um projeto simples de Post-it digital desenvolvido para organizar ideias, tarefas e lembretes de forma prática e intuitiva, mantendo os marcadores sempre acima da tela para não se perder em abas, e a guardando de forma simples.



Demonstração


Post-it inicial

Post-it

Assim que executamos o projeto, um post-it se abre na tela.

A janela possui propriedades para simular um post-it real:

  • Estático sobre todas as janelas, para evitar perda de foco
  • Pequeno, para anotações curtas
  • Coloridos, para melhor identificação visual
  • Simples, para maior eficiência

UX

No topo da janela temos 4 utilidades principais:

  • + -> Para instânciar um novo post-it
  • Titulo
  • - -> Para minimizar o post-it e diminuir a área de ocupação na tela
  • x -> Para fechar o post-it

Opções adicionais

Opções

Ao clicar com o botão direito sobre o post-it, um menu aparece com as seguintes opções:

  • Cores pasteis principais
  • Icones, que são exibidos quando o post-it é minimizado
  • Salvar/Salvar como, gera um arquivo .txt
  • Abrir, suporta arquivos .txt
  • Excluir, limpa toda a mensagem do post-it, e fecha caso n tenha nenhum conteúdo
  • Nova polaroid, abre uma nova instância de polaroid
  • Novo post-it, abre uma nova instância post-it

Varios Post-its

Instâncias

O post-it não é limitado a apenas um bloco de anotação, cada bloco tem sua cor e propósito diferentes, podendo ser tratados individualmente. As cores principais são:

  • Amarelo pastel
  • Translucido
  • Laranja pastel
  • Rosa pastel
  • Verde pastel
  • Azul pastel
  • Vermelho pastel

TopMost

O post-it possui também a propriedade de sobreposição, onde sempre ficará no topo evitando que a anotação saia de vista

Post-its Minimizados

Minimizar

Um post-it pode ser minimizado de duas formas, seja arrastando a janela até uma das bordas da tela, ou apertando no botão de minimizar no cabeçalho da janela.

Ao minimizar o post-it diminui seu tamanho, tornando-se um triangulo pequeno que pode ser amontoado nas beiradas da tela, evitando que fiquem atrapalhando a visão de algo importante.

Para sair do modo minimizado, basta clicar sobre o post-it

Polaroid

Essa tela serve mais como um quadro, mantendo as caracteristicas principais do post-it, minimizar, arrastar e sobrepor.

Pode ser usada para servir de referencia, enquanto armazena uma imagem, e uma legenda.

Diferente do post-it, pode ser redimensionada para uma melhor visualização, basta clicar sobre a ponta dobrada no canto inferior direito e arrastar.

⚙️ Executando o projeto

Pré-requisitos

  • Git
  • .NET SDK 8.0 ou superior
  • Visual Studio 2022

Instalação

  1. Clone o repositório.
  2. Entre na pasta do projeto.
  3. Restaure as dependências.
  4. Compile a aplicação.
  5. Execute o projeto.
git clone https://github.com/usuario/repositorio.git
 cd repositorio
 dotnet restore
 dotnet build
 dotnet run

🚀 Funcionalidades

  • Criação ilimitada de post-its independentes.
  • Janelas sempre visíveis utilizando o modo TopMost.
  • Personalização com diferentes cores e ícones.
  • Minimização inteligente para economizar espaço na área de trabalho.
  • Salvamento e carregamento de anotações em arquivos .txt.
  • Janela Polaroid para armazenar imagens com legenda.
  • Redimensionamento da Polaroid e movimentação livre das janelas.

📖 Como utilizar

  1. Execute a aplicação para criar o primeiro post-it.
  2. Digite suas anotações diretamente na janela.
  3. Utilize o botão + para criar novos post-its.
  4. Clique com o botão direito para acessar opções como alterar cores, salvar, abrir arquivos e criar Polaroids.
  5. Minimize as notas quando desejar liberar espaço na tela.
  6. Organize os post-its e Polaroids livremente pela área de trabalho.
  7. Salve suas anotações para continuar de onde parou em outra sessão.
