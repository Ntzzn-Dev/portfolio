Descrição

Um programa criado com intuito de estudo e conclusão do projeto de extensão de curso de Informática para negócios da Fatec. O Sistema serve como um suporte para vendas de um brechó.

O projeto conta com um banco de dados fake [Sem uso de SQL ou qualquer tipo de armazenamento de dados sofisticado] Possui um CRUD para quase todas as entidades, possibilitando um sistema completo no terminal.



Demonstração


Tela de login

Login

Apenas o funcionário possui acesso ao programa, tornando-se necessário apenas o login de funcionários.

Todo funcionário deverá fazer login usando o CPF e senha definidos na hora do cadastro.

Tela de login

Login

Assim que executamos o projeto, primeiro é necessário fazer o login.

A interface pede apenas o CPF e a senha, previamente definidos.

Ao entrar, o nome do funcionário é exibido, ele é quem vai ficar marcado ao executar vendas

Tela principal

Menu

A tela do menu exibe as principais entidades permitindo algumas ações padrões ao funcionário:

  • Create (Cadastro)
  • Read (Busca)
  • Update (Atualização)
  • Delete (Remoção)
  • List (Listagem)
